---
title: "offlinematch.conf"
---

离线玩家匹配规则的配置。在离线登录模块（`auth-offline`）激活时有效。

```hocon
# 高级设定
advanced {}
# ⚠️是否允许进行匹配
enable=true
# ⚠️Host匹配设定
host-match {
    start=[
        offline,
        o-
    ]
}
# UUID匹配设定
uuid-match {
    # ⚠️是否允许默认uuid生成方法 匹配为离线
    offline=true
    # 关于PCL2启动器匹配的细节设定
    pcl2 {
        # PCL2的UUID匹配
        enable=true
        # PCL2的UUID进行哈希计算匹配
        hash=true
        # PCL2的苗条模型UUID匹配
        slim=true
    }
    # 是否允许全0的UUID(Zalith) 匹配为离线
    zero=true
}

```
