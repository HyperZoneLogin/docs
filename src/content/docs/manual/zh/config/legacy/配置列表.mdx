本页面介绍插件所涉及的各种配置文件。

:::caution[配置格式已过时！]
此页面所载的配置格式已经过时，仅适用于26.4.5-SNAPSHOT-8fcde1ec及以前版本。  
关于目前正在使用的配置格式，请前往[新版配置列表](/manual/zh/config/配置列表/)页面。
:::

:::danger[危险]
**部分配置项会直接影响本插件的关键行为，配置不得当的情况下很可能导致插件无法正常工作乃至崩溃。**  
此类配置项已在对应的描述中添加了“⚠️”作为标识。除非你真的知道你在做什么，否则不要改动这些配置项目。
:::
