---
title: "modules.conf"
---

控制插件内置模块的配置文件。若同名外部插件已安装，则自动跳过内置版本。

```hocon
# 是否启用内置离线认证模块；若同时安装 hzl-auth-offline 外部插件，则自动跳过内置版本
auth-offline=true
# 是否启用内置 Yggdrasil 认证模块；若同时安装 hzl-auth-yggd 外部插件，则自动跳过内置版本
auth-yggd=true
# 是否启用内置数据迁移模块；默认关闭，按需开启。若同时安装 hzl-data-merge 外部插件，则自动跳过内置版本
data-merge=false
# 是否启用内置皮肤缓存模块；若同时安装 hzl-profile-skin 外部插件，则自动跳过内置版本
profile-skin=true
# 是否启用内置安全防护模块；若同时安装 hzl-safe 外部插件，则自动跳过内置版本
safe=true
```
